=============
Installation
=============

`````````````````
Setup (Important)
`````````````````
.. code-block:: console

   $ pip3 install 'setuptools<=66'
   $ pip3 install 'wheel<=0.38.4'

`````````````````
Install package
`````````````````

#. Using PyPI:

   .. code-block:: console

      $ pip install opcc

#. Directly from source (recommended):

   .. code-block:: console

       $ pip install git+https://github.com/koulanurag/opcc@main#egg=opcc
